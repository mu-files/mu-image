# mu-image

Python utilities for working with raw image data, including DNG file generation and processing.

## Package

This repository contains the `muimg` Python package. See [muimg/README.md](muimg/README.md) for detailed documentation.

## Installation

```bash
pip install muimg
```

Or for development:

```bash
git clone https://github.com/mu-files/mu-image.git
cd mu-image/muimg
pip install -e .
```
